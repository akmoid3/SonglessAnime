# Songless Anime ??

A fun guessing game built with Angular and Tailwind CSS! 

**How to Play:**
1. A random Anime song (mp3) is selected.
2. You get to listen to increasingly longer snippets of the song (0.1s, 1.1s, 2.1s, 3.1s, 4.1s).
3. If you guess wrong or skip, you move to the next level.
4. Try to guess the Anime before running out of attempts (max 5 tries)!

## Development Server
Run \
pm start\ or \
g serve\ to start the development server. Navigate to \http://localhost:4200/\.

## Configuration
Currently, dummy URLs are used in \src/app/services/song.ts\. Replace these with valid mp3/webm sources to hear actual Anime songs.
